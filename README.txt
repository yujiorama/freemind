Freemind を Macos 上でアプリケーションに見せかけるためのフォルダ構成です。

以下の手順で Finder からアプリケーションとして識別されるようになります。

1. このリポジトリを取得して、Freemind.app という名前に変更します
2. Freemind活用クラブさん(http://www.freemind-club.com/download.htm)から
   全 OS 用最大構成実行ファイルをダウンロードします
3. ダウンロードした zip ファイルを展開します
4. 展開したファイルを Freemind.app/Contents/MacOS の下にコピーします
5. Terminal を起動して、"chmod +x Freemind.app/Contents/MacOS/freemind.sh" と実行します
6. Finder から Freemind.app を実行します
7. おそらく JRE のアップデートを要求されるので従います
8. JRE のアップデートが完了したら、全ての手順は完了です。

enjoy Freemind !
